# Clean-the-Netflix-Dataset
I will load the Netflix Movies and TV Shows dataset from Kaggle and clean it using Pandas. The dataset has missing values, wrong data types, and mixed-type columns — exactly the kind of mess you find in real data.  The goal is not just to drop nulls, but to understand why values are missing and make deliberate decisions about each column.
